# Donna Domésticas - Landing Page

Landing page para a campanha **Operação Casa Organizada** com evento online 100% gratuito.

## 📋 Estrutura do Projeto

```
.
├── index.html           # Landing page principal
├── css/                 # Estilos (se necessário adicionar depois)
├── assets/              # Imagens e recursos (se necessário adicionar depois)
└── README.md           # Este arquivo
```

## 🚀 Como usar

1. **Localmente**: Abra `index.html` diretamente no navegador
2. **Via servidor**: 
   ```bash
   python3 -m http.server 8000
   # ou
   npx http-server
   ```
3. **Deploy**: Faça upload de todos os arquivos para seu hosting
